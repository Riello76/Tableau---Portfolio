# Global Health Insights - Tableau Project

This Tableau project provides visual insights into global health data. The dataset includes information on life expectancy, population, BMI, and gender distribution, which have been visualized through multiple interactive dashboards.

## Dashboards Included

1. **Life Expectancy by Continent**
   - Shows average life expectancy for each continent.
   - Useful for comparing health outcomes across regions.

2. **Life Expectancy vs BMI**
   - Analyzes the relationship between life expectancy and average Body Mass Index (BMI).
   - Helps explore correlations between health indicators and longevity.

3. **Population by Country â€“ 2008**
   - Displays country-level population data for the year 2008.
   - Useful for demographic analysis and comparison between countries.

4. **Life Expectancy Trend**
   - Shows changes in life expectancy over time.
   - Allows historical trend analysis and country comparisons.

5. **Population by Gender**
   - Breaks down population statistics by gender.
   - Useful for understanding gender distribution globally.
