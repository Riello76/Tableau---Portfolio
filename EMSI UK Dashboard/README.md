# EMSI UK Tableau Dashboard Project

This project visualizes employment and industry data for the UK using **Tableau**. The dataset is provided in the accompanying Excel file and has been used to create interactive dashboards highlighting key trends in employment by industry, sub-industry, and city.

## Dashboards Included

1. **Top 10 Industries**
   - Displays the 10 industries with the highest employment numbers across the UK.
   - Allows quick identification of the most significant employment sectors.

2. **Average Percentage by City**
   - Shows the average percentage distribution of jobs across different cities.
   - Highlights regional differences in employment distribution.

3. **Top 5 Sub-Industries**
   - Focuses on the top 5 sub-industries within the largest industries.
   - Useful for analyzing specialized employment sectors.

4. **2014 Jobs by City**
   - Presents the total number of jobs per city for the year 2014.
   - Enables historical comparison and city-level analysis.

## Key Insights

- Identifies leading industries and sub-industries driving employment in the UK.
- Highlights regional variations in job distribution.
- Provides a historical view of city-level employment trends.

## Usage

- Open the provided Excel file in Tableau to recreate or explore dashboards.
- Use filters and interactive elements in Tableau to analyze trends by industry, sub-industry, and city.
- Ideal for workforce analysis, economic studies, and visualization practice.

## File Information

- **Dataset Filename:** `EMSI_UK_Data.xlsx`
- **Format:** Excel (.xlsx)
- **Contents:** Employment and industry data by city, industry, sub-industry, and year (2014).
