# Top 10 Spotify Genres by Popularity - Tableau Project

This Tableau project visualizes the most popular music genres on Spotify. The dataset contains information on genre popularity, which is displayed through an interactive dashboard.

## Dashboard Included

1. **Top 10 Genres**
   - Highlights the ten most popular music genres on Spotify.
   - Shows overall popularity ranking.

## Key Insights

- Quickly identifies the most listened-to genres on Spotify globally.
- Provides a clear view of which genres dominate in popularity.
